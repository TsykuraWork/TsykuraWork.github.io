    2. Исследование эффективности архитектур свёрточных нейронных сетей в решении задачи классификации при малой обучающей выборке

      Авторы: В.М. Рассолова, Е.Д. Позднякова, В.В. Лотырева

      Описание: В статье представлен обзор на наиболее известные архитектуры свёрточных нейронных сетей, а также их сравнительная характеристика. Проведен анализ разработанной уникальной архитектуры и компаративный разбор. При использовании различных архитектур CNN была получена высокая точность классификации во всех исследованных случаях. Также был проведен анализ влияния гиперпараметров и их оптимального выбора на эффективность работы CNN.

      Источник: В.М. Рассолова, Е.Д. Позднякова, В.В. Лотырева. Исследование эффективности архитектур свёрточных нейронных сетей в решении задачи классификации при малой обучающей выборке / Auditorium 2023 №2 http://www.irbis-nbuv.gov.ua ...

      Авторы: А.А. Бабенко

      Источник: А.А. Бабенко. Об особенностях сверточной архитектуры нейросети / Мировая наука 2020 №12, с. 114-137, http://www.mathnet.ru ...

    4. Разработка защищенного протокола обмена сообщениями между пользователями и клиент-серверного программного обеспечения Crypto Messenger

      Авторы: А.И. Крушанов, А.В. Чернышова

      Описание: В данной работе протокол обмена мгновенными сообщениями на базе протокола XMPP, сформированы требования к клиент-серверному программному продукту Crypto Messenger. Проведено проектирование и реализация программного продукта.

      Источник: Программная инженерия: методы и технологии разработки информационно-вычислительных систем (ПИИВС-2018) / Материалы II Международной научно-практической конференции. – Донецк, ДонНТУ – 2018.

    5. Исследование нейронных систем для распознавания текстовых изображений

      Авторы: А.А. Морозов

      Описание: В статье проводится исследование нейронных систем для распознавания текстовых изображений. Рассматриваются понятие, структура и принцип работы нейронных сетей. Приводятся классификация нейронных сетей по способу обучения и количеству слоёв и этапы процесса их обучения. Выявляются характерные черты, отличия и специфика работы свёрточных нейронных сетей, чаще всего использующихся для распознавания текстовых изображений.

      Источник: А.А. Морозов. Исследование нейронных систем для распознавания текстовых изображений / Вестник Астраханского государственного технического университета № 1, Астрахань, 2008, с. 87-92, https://cyberleninka.ru ...
